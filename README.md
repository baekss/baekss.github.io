# baekss.github.io
This study provides an approach to apply AOP to event handler function of elements.

この見本はelementのevent handler functionにAOPを適用したことでproxyパターンが活用されました。

해당 코드는 요소의 이벤트 핸들러 함수에 프록시 패턴을 이용하여 AOP를 적용하였습니다.
